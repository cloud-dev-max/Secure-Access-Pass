### Issue #3: Occupancy Math ✅ Already Correct
**Location:** `app/api/occupancy/route.ts` line 51  
**Status:** Math was already correct, verified in V7.2  
**Code:**
```typescript
const totalOccupancy = (residentsCount || 0) + accompanyingGuests + (visitorPassesCount || 0)
```
**Formula:**
- `residentsCount`: Count of profiles with `current_location='INSIDE'`
- `accompanyingGuests`: Sum of `active_guests` from all residents inside
- `visitorPassesCount`: Count of guest passes with `status='used'` today
- **Total = Residents + Guests + Visitors** ✅

### Issue #6: Revenue Loading Bug ✅ FIXED in V7.2
**Location:** `app/dashboard/page.tsx` lines 882-892  
**Problem:** Infinite spinner when `revenueData` was `null`  
**Solution:**
```typescript
// OLD (V7.1)
{revenueData ? ( ... ) : ( <Loader2 /> )}

// NEW (V7.2)
{revenueLoading ? (
  <Loader2 className="animate-spin" />
) : revenueData?.daily && revenueData.daily.length > 0 ? (
  <span>${revenueData.daily[0]?.revenue?.toFixed(0) || '0'}</span>
) : (
  <span className="text-gray-400">$0</span>
)}
```
**Result:** Three states handled correctly:
1. Loading: Shows spinner
2. Data loaded: Shows revenue amount
3. No data: Shows $0 (not spinner)

```typescript
let bgColor = 'bg-gray-50'

if (log.result === 'DENIED') {

  borderColor = 'border-red-200'
  iconColor = 'text-red-600'

  bgColor = 'bg-green-50'
  borderColor = 'border-green-200'
  iconColor = 'text-green-600'

}
```
