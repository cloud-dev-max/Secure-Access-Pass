### **Bug #3: Resident Portal Occupancy Undercounting (Missing Guests)** ✅
**Severity:** 🔴 Critical  
**Impact:** Residents see incorrect facility occupancy

**Root Cause:**
- `/api/facility-status` only counted resident profiles INSIDE
- Did not include accompanying guests (`active_guests`)
- Did not include active visitor passes
- Mismatch with manager dashboard logic

**Fix Applied:**
- ✅ Updated `app/api/facility-status/route.ts` to use same math as `/api/occupancy`
- ✅ Count residents INSIDE
- ✅ Sum `active_guests` for those residents
- ✅ Count active visitor passes (status='used', not expired)
- ✅ Return total as single `current_occupancy` integer

**Code Changes:**
```typescript
// app/api/facility-status/route.ts lines 29-77
// 1. Count residents currently INSIDE
const { count: residentsInsideCount, error: residentsError } = await adminClient
  .from('profiles')
  .select('*', { count: 'exact', head: true })
  .eq('property_id', propertyId)
  .eq('role', 'resident')
  .eq('current_location', 'INSIDE')
  .eq('is_active', true)

const residentsInside = residentsInsideCount || 0
console.log('🏊 Residents INSIDE:', residentsInside)

// 2. Sum active_guests from residents who are INSIDE
const { data: insideResidents, error: guestsError } = await adminClient
  .from('profiles')
  .select('active_guests')
  .eq('property_id', propertyId)
  .eq('role', 'resident')
  .eq('current_location', 'INSIDE')
  .eq('is_active', true)

const accompanyingGuests = (insideResidents || []).reduce(
  (sum, r) => sum + (r.active_guests || 0),
  0
)
console.log('👥 Accompanying Guests:', accompanyingGuests)

// 3. Count active visitor passes (used status, not expired)
const { count: visitorPassCount, error: visitorError } = await adminClient
  .from('guest_passes')
  .select('*', { count: 'exact', head: true })
  .eq('property_id', propertyId)
  .eq('status', 'used')
  .gt('expires_at', new Date().toISOString())

const visitorPasses = visitorPassCount || 0
console.log('🎫 Active Visitor Passes:', visitorPasses)

// 4. Calculate total occupancy
const totalOccupancy = residentsInside + accompanyingGuests + visitorPasses
console.log('📊 Total Occupancy:', totalOccupancy)
```

### **Bug #5: Guest Pass Creation Crash (Line 348)** ✅
**Severity:** 🔴 Critical  
**Impact:** App crashes when guest pass creation fails

**Root Cause:**
- Frontend threw error without catching API response
- No error state to display API error messages
- User saw generic crash instead of helpful error message
- Missing error parsing from `/api/guest-passes` response

**Fix Applied:**
- ✅ Added `guestPassError` state variable
- ✅ Parse API response JSON for `error` or `details` fields
- ✅ Display red error banner above form
- ✅ Prevent form reset on error
- ✅ Clear error on successful submission
- ✅ Added try-catch for network errors

**Code Changes:**
```typescript
// app/resident/page.tsx lines 60, 319-370
const [guestPassError, setGuestPassError] = useState('') // V7.3: Error handling

const createGuestPass = async (e: React.FormEvent) => {
  e.preventDefault()
  if (!resident) return

  setGuestPassError('') // Clear previous errors

  // Check guest pass limit
  const activeGuestPasses = guestPasses.filter(p => {
    const isActive = p.status === 'active'
    const notExpired = new Date(p.expires_at) > new Date()
    return isActive && notExpired
  })
  if (activeGuestPasses.length >= maxGuestsAllowed) {
    setGuestPassError(`You have reached the maximum of ${maxGuestsAllowed} active guest passes.`)
    return
  }

  setCreatingPass(true)

  try {
    const response = await fetch('/api/guest-passes', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        purchased_by: resident.id,
        guest_name: guestName || null,
        guest_email: guestEmail || null,
        guest_phone: guestPhone || null,
      }),
    })

    if (!response.ok) {
      // V7.3: Parse the actual error from API
      const errorData = await response.json()
      const errorMessage = errorData.error || errorData.details || 'Failed to create guest pass'
      setGuestPassError(errorMessage)
      return
    }

    // Success - Reset form
    setGuestName('')
    setGuestEmail('')
    setGuestPhone('')
    setGuestPassError('')
    setShowGuestPassForm(false)

    await loadGuestPasses(resident.id)
  } catch (error) {
    console.error('Error creating guest pass:', error)
    setGuestPassError('Network error. Please check your connection and try again.')
  } finally {
    setCreatingPass(false)
  }
}
```
