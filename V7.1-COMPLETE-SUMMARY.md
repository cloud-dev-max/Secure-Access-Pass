## Migration Requirements

**CRITICAL: Run this migration before deploying:**

```sql
-- migrations/0007_v7.1_personal_limits.sql (already exists)

-- Add personal_guest_limit to profiles (if not exists)
ALTER TABLE profiles 
ADD COLUMN IF NOT EXISTS personal_guest_limit INTEGER DEFAULT NULL;

-- Add max_visitor_passes to properties (if not exists)
ALTER TABLE properties 
ADD COLUMN IF NOT EXISTS max_visitor_passes INTEGER DEFAULT 10;

-- Add last_scan_at index for performance
CREATE INDEX IF NOT EXISTS idx_profiles_last_scan_at 
ON profiles(last_scan_at DESC);

-- Add expires_at index for guest passes
CREATE INDEX IF NOT EXISTS idx_guest_passes_expires_at 
ON guest_passes(expires_at);
```

**Run in Supabase SQL Editor:**
1. Go to https://supabase.com/dashboard
2. Select your project → SQL Editor → New Query
3. Paste the migration SQL above
4. Click **Run**
5. Verify: `SELECT * FROM properties LIMIT 1;` should show `max_visitor_passes` column
