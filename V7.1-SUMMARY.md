## 💡 Quick Fixes Summary

### **Fix #1: Guest Pass Bug** (Most Important!)

**Problem:** Expired passes still count as active

**Solution:** Filter passes properly:
```typescript
// Only count ACTIVE and NOT EXPIRED
Active passes: {guestPasses.filter(p => {
  const isActive = p.status === 'active'
  const notExpired = new Date(p.expires_at) > new Date()
  return isActive && notExpired
}).length} / {maxGuestsAllowed} allowed
```

---

### **Fix #2: Occupancy Bug**

**Problem:** Dashboard shows 0/50 even after check-ins

**Solution:** Verify `app/api/check-access/route.ts` updates `current_location`:
```typescript
await adminClient
  .from('profiles')
  .update({ 
    current_location: scan_type === 'ENTRY' ? 'INSIDE' : 'OUTSIDE'
  })
  .eq('id', resident.id)
```
